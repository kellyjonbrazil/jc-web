About
Welcome to the jc demo web app! jc is a command-line utility and Python library that converts the output of many commands and file-types to JSON for easier parsing in scripts. jc has been hand-crafted by Kelly Brazil. (kellyjonbrazil@gmail.com) This web app is based on jc version 1.13.4. For more information, please see the jc project github site.
