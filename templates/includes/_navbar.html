About

Welcome to the jc demo web app!

jc is a command-line utility and Python library that converts the output of many commands and file-types to JSON for easier parsing in scripts.

jc is hand-crafted with love by Kelly Brazil.

This web app is based on jc version {{ jc_info.version }}.

For more information, please see the jc project github site.
